培养方案查询

这里收录了钱学森学（书）院各专业班级的培养方案，主要包括各专业方向在不同学期所要学习的课程——特别是基础课程与专业课程，并与课程中心的相关页面关联。

各专业的培养方案信息均已在西安交通大学综合教学服务平台的“全校方案查询”模块发布。

理科与医学实验班

  • 数学试验班 Mathematics Expr. Class
  • 物理试验班 Physics Expr. Class
  • 化生试验班 Chemitry & Biology Expr. Class
  • 计算机试验班 Computer Science Expr. Class
  • 人工智能试验班 Artificial Intelligence Expr. Class
  • 侯宗濂医学试验班 Hou Zonglian Medicine Expr. Class

越杰班

越杰班是大一入校时即划分好专业。大一时全班同上公选课，大二后培养方案中将加入本专业的相关课程。

  • 越杰班（公选课） Acadamic Elite Program (Shared)
  • 越杰班-电气 Acadamic Elite Program (Electrical Engineering)
  • 越杰班-自动化 Acadamic Elite Program (Automation)
  • 越杰班-计算机 Acadamic Elite Program (Computer Science)
  • 越杰班-材料 Acadamic Elite Program (Materials)
  • 越杰班-能动 Acadamic Elite Program (Energy & Power Engineering)
  • 越杰班-机械 Acadamic Elite Program (Mechanical Engineering)
  • 越杰班-力学 Acadamic Elite Program (Mechanics)

钱学森班

钱学森班在大一年级时上相同的通识课；大二开始前，根据志愿、参考大一一年公选课的均分排名，划分具体的专业。

  • 钱学森班（公选课） Qian Xuesen Expr. Program (Shared)
  • 钱学森班-电气 Qian Xuesen Expr. Program (Electrical Engineering)
  • 钱学森班-能动 Qian Xuesen Expr. Program (Energy & Power Engineering)
  • 钱学森班-机械 Qian Xuesen Expr. Program (Mechanical Engineering)
  • 钱学森班-计算机 Qian Xuesen Expr. Program (Computer Science)
  • 钱学森班-自动化 Qian Xuesen Expr. Program (Automation)
  • 钱学森班-微电子 Qian Xuesen Expr. Program (Microelctronics)
  • 钱学森班-力学 Qian Xuesen Expr. Program (Mechanics)

少年班

16 级及以后的少年班，根据成绩排名、个人志愿、面试结果分流到以上各试验班或钱院以外专业班级，方案不再单列。13 – 15 级少班各专业培养方案与钱学森班相近。

  • 少年班（预科二年级） Honors Youth Program (Grade 2, Prep.)
  • 少年班（大二年级公选课） Honors Youth Program (Shared, Sophomore)
  • 少年班-电气 Honors Youth Program (Electrical Engineering)
  • 少年班-自动化 Honors Youth Program (Automation)
  • 少年班-微电子 Honors Youth Program (Microelctronics)
  • 少年班-力学 Honors Youth Program (Mechanics)
